README


This repository is a submission for the final course project for the "Getting and Cleaning Data"
course offered by Johns Hopkins University and hosted through Coursera. Included are the following
files:

1. run_analysis.R
        
        A script that applies the set of instructions defined by the project to the data set
        downloaded from 
        https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip.
        
2. CompletedProject.csv
        
        A .csv file of the output of the tidy data set created by run_analysis.R.
        
3. codebook2.txt
        
        A step-by-step guide describing the steps used in run_analysis.R, the variables/data 
        frames defined within, and any adjustments made to those data frames in the creation
        of CompletedProject.csv.
